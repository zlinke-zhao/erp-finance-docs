🚨 未匹配预警规则

预警触发条件
1. 入库单生成后 7天 未匹配发票
2. 发票审核通过后 15天 货物未验收
3. 同一物流跟踪号匹配 多张发票（金额超入库单）
4. 发票金额 > 入库单金额 10% 以上
预警处理流程
1. 系统每日凌晨触发预警任务
2. 生成预警清单推送至财务负责人
3. 财务核实原因（发票未到/物流延迟/金额错误）
4. 手动标记预警状态（已处理/待跟进/无需处理）
5. 超30天未处理预警升级至财务总监

▼

⚠️ 汇率异常处理规则

异常场景
1. 当日汇率同步失败（数据源不可用）
2. 汇率波动超过 ±5%（与前一日比）
3. 手动调整汇率幅度超过 ±2%
4. 历史汇率查询无数据
异常处理
1. 同步失败: 自动使用备用数据源 → 仍失败则触发人工介入
2. 汇率大幅波动: 触发财务负责人审批 + 备注原因
3. 手动调整超限: 需财务总监审批 + 留存审批记录
4. 无历史汇率: 取最近有效汇率 + 标记"汇率预估"
