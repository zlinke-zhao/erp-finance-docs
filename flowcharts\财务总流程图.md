# ERP财务管理系统 - 业务流程图

## 流程一: 销售收款全流程（销售订单 → 总账）

```mermaid
flowchart TD
    subgraph Sales["📦 销售收款业务流程"]
        direction TB
        
        SO["<b>1. 销售订单<br/>Sales Order</b><br/>客户/商品/数量/单价<br/>含税金额/税率"]
        
        DN["<b>2. 送货单<br/>Delivery Note</b><br/>关联销售订单<br/>实际发货数量<br/>仓库/批次/日期"]
        
        AR["<b>3. 应收对账<br/>AR Reconciliation</b><br/>确认应收账款<br/>对账单生成<br/>客户确认签字"]
        
        INV["<b>4. 税务开票<br/>Tax Invoicing</b><br/>增值税专用发票<br/>开票信息核验<br/>税控系统对接"]
        
        ENTRY["<b>5. 发票录入<br/>Invoice Entry</b><br/>发票信息录入系统<br/>校验码/号码匹配<br/>与应收单据关联"]
        
        RCPT["<b>6. 实收收款<br/>Cash Receipt</b><br/>银行到账确认<br/>收款方式登记<br/>预收款核销"]
        
        BILL["<b>7. 票据往来<br/>Bill Management</b><br/>承兑汇票管理<br/>票据贴现/背书<br/>到期提醒"]
        
        GL["<b>8. 总账凭证<br/>General Ledger</b><br/>自动生成凭证<br/>借贷平衡校验<br/>科目账簿登记"]
        
        SO -->|"发货执行"| DN
        DN -->|"确认交付"| AR
        AR -->|"开具发票"| INV
        INV -->|"录入系统"| ENTRY
        ENTRY -->|"收到款项"| RCPT
        RCPT -->|"票据结算"| BILL
        BILL -->|"记账入账"| GL
        
        subgraph Checkpoints["✅ 关键控制点"]
            C1["🔍 订单审批"]
            C2["🔍 发货核对"]
            C3["🔍 对账确认"]
            C4["🔍 发票校验"]
            C5["🔍 收款核销"]
            C6["🔍 凭证审核"]
        end
        
        SO -.->|审批| C1
        DN -.->|核对| C2
        AR -.->|确认| C3
        INV -.->|校验| C4
        RCPT -.->|核销| C5
        GL -.->|审核| C6
    end
    
    subgraph DataFlow["📊 数据流向"]
        direction LR
        D1["销售数据"] --> D2["库存扣减"]
        D2 --> D3["应收增加"]
        D3 --> D4["税务数据"]
        D4 --> D5["现金流入"]
        D5 --> D6["总账更新"]
    end

    style Sales fill:#e8f5e9,stroke:#2e7d32,stroke-width:2px
    style Checkpoints fill:#fff3e0,stroke:#ef6c00,stroke-width:1px,stroke-dasharray: 5 5
    style DataFlow fill:#e3f2fd,stroke:#1565c0,stroke-width:1px
    style SO fill:#c8e6c9,stroke:#388e3c,stroke-width:2px
```

## 流程二: 采购付款全流程（采购申请 → 总账）

```mermaid
flowchart TD
    subgraph Purchase["🛒 采购付款业务流程"]
        direction TB
        
        PR["<b>1. 采购申请<br/>Purchase Request</b><br/>需求部门发起<br/>物料/规格/预算<br/>审批流程流转"]
        
        PO["<b>2. 采购订单<br/>Purchase Order</b><br/>供应商选择/比价<br/>交期/价格条款<br/>合同编号关联"]
        
        RV["<b>3. 收货验收<br/>Goods Receipt</b><br/>质检合格确认<br/>实收数量记录<br/>入库单生成"]
        
        AP["<b>4. 应付对账<br/>AP Reconciliation</b><br/>供应商对账单<br/>采购入库核对<br/>差异处理"]
        
        VIN["<b>5. 进项发票<br/>Input VAT Invoice</b><br/>供应商专票收取<br/>发票真伪查验<br/>认证抵扣申报"]
        
        VENTRY["<b>6. 发票录入<br/>Invoice Entry</b><br/>进项发票信息录入<br/>三单匹配校验<br/>应付账款挂账"]
        
        PAY["<b>7. 付款执行<br/>Payment Execution</b><br/>付款计划安排<br/>银行转账/支票<br/>预付款核销"]
        
        PBILL["<b>8. 票据往来<br/>Payable Bill Mgmt</b><br/>应付票据管理<br/>承兑汇票开出<br/>到期兑付跟踪"]
        
        PGL["<b>9. 总账凭证<br/>General Ledger</b><br/>自动生成凭证<br/>进项税额确认<br/>成本/费用归集"]
        
        PR -->|"审批通过"| PO
        PO -->|"供应商发货"| RV
        RV -->|"入库确认"| AP
        AP -->|"索取发票"| VIN
        VIN -->|"录入系统"| VENTRY
        VENTRY -->|"安排付款"| PAY
        PAY -->|"票据结算"| PBILL
        PBILL -->|"记账入账"| PGL
        
        subgraph PCheckpoints["✅ 采购控制点"]
            PC1["🔍 预算控制"]
            PC2["🔍 价格审核"]
            PC3["🔍 质检验收"]
            PC4["🔍 三单匹配"]
            PC5["🔍 付款审批"]
            PC6["🔍 成本核算"]
        end
        
        PR -.->|预算| PC1
        PO -.->|比价| PC2
        RV -.->|质检| PC3
        VENTRY -.->|匹配| PC4
        PAY -.->|审批| PC5
        PGL -.->|核算| PC6
    end
    
    subgraph TaxFlow["💰 税务抵扣链路"]
        direction LR
        T1["进项发票收取"] --> T2["发票认证"]
        T2 --> T3["税额抵扣"]
        T3 --> T4["成本入账"]
        T4 --> T5["进项转出(如需)"]
    end
    
    style Purchase fill:#fff8e1,stroke:#f9a825,stroke-width:2px
    style PCheckpoints fill:#fce4ec,stroke:#c62828,stroke-width:1px,stroke-dasharray: 5 5
    style TaxFlow fill:#f3e5f5,stroke:#7b1fa2,stroke-width:1px
    style PR fill:#ffecb3,stroke:#ffa000,stroke-width:2px
    style PGL fill:#ffe082,stroke:#f57f17,stroke-width:3px
```
