1. 模块概述

1.1 业务定位

票据往来模块是ERP财务管理中非现金收款的核心枢纽，负责管理商业汇票、支票、本票等票据从收取到最终处置的全生命周期。 ` ┌──────────────────────────────────────────────────────────┐ │ 票据模块架构定位 │ ├──────────────────────────────────────────────────────────┤ │ │ │ 上游: 实收收款模块 ──── 收到票据 ────→ 票据往来模块 │ │ │ │ │ ┌──────────┼──────────┐ │ │ ↓ ↓ ↓ │ │ 背书转让 贴现/质押 到期兑付 │ │ │ │ │ │ │ 下游: ↓ ↓ ↓ │ │ ┌──────────────┐ ┌──────────┐ ┌──────────┐ │ │ │ 采购付款模块 │ │ 银企对接 │ │ 总账凭证 │ │ │ │ (应付核销) │ │ (资金) │ │ (入账) │ │ │ └──────────────┘ └──────────┘ └──────────┘ │ │ │ └──────────────────────────────────────────────────────────┘ `

    4.4 票面要素校验规则

    `python def validate_bill_note(bill: dict) -> list[str]: """票面要素完整性校验，返回错误列表""" errors = [] # 必填字段 required = ['bill_type', 'bill_number', 'draft_date', 'due_date', 'face_amount', 'drawer', 'acceptor'] for field in required: if not bill.get(field): errors.append(f"缺少必填字段: {field}") # 业务规则校验 if bill.get('face_amount', 0) <= 0: errors.append("票面金额必须大于0") if bill.get('due_date') and bill.get('draft_date'): if bill['due_date'] <= bill['draft_date']: errors.append("到期日期必须晚于出票日期") days = (bill['due_date'] - bill['draft_date']).days if days > 365: errors.append("票据期限不得超过一年") # 银行承兑汇票必须填写承兑银行 if bill.get('bill_type') == 1 and not bill.get('acceptor_bank'): errors.append("银行承兑汇票必须填写承兑银行") # 金额一致性（OCR场景） if bill.get('amount_chinese') and bill.get('face_amount'): # 大写金额与小写金额比对 if not amount_match(bill['amount_chinese'], bill['face_amount']): errors.append("大写金额与小写金额不一致") return errors `

    4.5 贴现计算规则

    `python def calculate_discount( face_amount: float, # 票面金额 discount_rate: float, # 贴现年利率 (例: 0.032 = 3.2%) discount_date: date, # 贴现日期 due_date: date # 到期日期 ) -> dict: """ 贴现利息计算 公式: 贴现利息 = 票面金额 × 贴现利率 × 贴现天数 / 360 """ discount_days = (due_date - discount_date).days discount_interest = round(face_amount discount_rate discount_days / 360, 2) actual_amount = round(face_amount - discount_interest, 2) return { 'face_amount': face_amount, 'discount_rate': discount_rate, 'discount_days': discount_days, 'discount_interest': discount_interest, 'actual_amount': actual_amount, } ` ---
