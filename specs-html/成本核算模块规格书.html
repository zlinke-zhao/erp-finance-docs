1. 模块定位与业务目标

1.1 一句话定义

成本核算模块是ERP系统中"归集-分摊-核算-结转"闭环的成本管控引擎，负责从直接材料/直接人工/制造费用归集开始，经过多维度分摊、在制品核算、完工产品成本计算，最终生成成本结转凭证写入总账的全流程管理。

1.4 核心业务目标

目标指标当前痛点
成本核算及时性月结后3个工作日内完成成本核算手工统计耗时2周，数据滞后
成本准确性实际成本与标准成本差异率不超过5%分摊基准不合理，成本失真
在制品核算在制品估值误差不超过3%约当产量手工估算，偏差大
成本可追溯每笔成本可追溯到原始单据费用归集链路断裂，无法追溯
差异可分析量差/价差自动拆分只看总额差异，无法定位原因

6. 关键业务规则

6.1 成本核算方法

方法适用场景核算逻辑
品种法大量单步骤生产按产品品种归集+分配
分批法单件小批生产按生产订单/批号归集
分步法大量多步骤生产按生产步骤逐步结转
作业成本法间接费用占比高按作业动因分摊制费

6.2 在制品核算方法

方法计算逻辑适用场景
约当产量法期末在制品数量 x 完工程度 = 约当产量，再按比例分配成本均匀投料、加工进度可测
定额成本法期末在制品按定额成本计价，实际成本与定额差异由完工产品承担定额管理完善
不计算在制品当期发生的成本全部由完工产品承担在制品数量少或稳定

6.3 制造费用分摊基准

分摊基准计算方式适用场景
生产工时制费总额 x 某产品工时/总工时人工密集型
机器工时制费总额 x 某产品机时/总机时设备密集型
直接材料成本制费总额 x 某产品材料/总材料材料相关制费
直接人工成本制费总额 x 某产品人工/总人工人工相关制费
产品产量制费总额 x 某产品产量/总产量单一产品或类似产品
固定比例按预设比例分摊无法确定单一基准

6.4 成本结转顺序


1. 材料费用归集 → CT01
2. 人工费用归集 → CT02  
3. 制造费用归集 → CT03
4. 辅助生产分配 → CT05
5. 制造费用分摊 → CT04
6. 在制品核算 → 约当产量计算
7. 完工产品入库 → CT06
8. 销售成本结转 → CT07
9. 成本差异处理 → CT08
10. 月末结账

⚠️ 结转顺序不可逆，前一步未完成不可执行下一步

6.5 差异分析公式

差异类型计算公式
材料量差(实际用量 - 标准用量) x 标准单价
材料价差(实际单价 - 标准单价) x 实际用量
人工效率差异(实际工时 - 标准工时) x 标准工资率
工资率差异(实际工资率 - 标准工资率) x 实际工时
制费耗费差异实际制费 - 预算制费(按实际产量调整)
制费效率差异(实际工时 - 标准工时) x 标准分摊率

安全成本数据按部门权限隔离，非授权人员不可查看

10. 验收标准

验收项标准
成本归集材料/人工/制费三要素自动归集率不低于90%，可追溯到原始单据
制费分摊支持至少5种分摊基准，分摊试算结果与正式结果一致
在制品核算约当产量法+定额法双模式，与手工核算偏差不超过1%
完工结转产品单位成本计算正确，完工入库+销售成本自动结转
差异分析量差/价差自动拆分，差异率超5%自动预警
凭证生成8套凭证模板100%自动生成，结转顺序严格受控
报表分析6类成本报表可查，支持按产品/部门/期间多维度
