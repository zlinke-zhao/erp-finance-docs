1. 模块概述

1.1 业务定位

税务管理模块是ERP财务系统的税务数据中心，负责税务基础配置、税金自动计提、税务申报管理、税负分析等全流程税务管理工作。它与总账、应收应付、发票管理模块深度集成，实现业财税一体化。 `` ┌─────────────────────────────────────────────────────────────────┐ │ 🏛️ 税务管理模块 │ ├─────────────────────────────────────────────────────────────────┤ │ ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ │ │ │ 税务基础 │ │ 税金计提 │ │ 税务申报 │ │ 税负分析 │ │ │ │ 配置 │ │ │ │ │ │ │ │ │ └────┬─────┘ └────┬─────┘ └────┬─────┘ └────┬─────┘ │ │ │ │ │ │ │ │ └─────────────┴─────────────┴─────────────┘ │ │ │ │ │ ┌─────┴─────┐ │ │ ▼ ▼ │ │ ┌──────────┐ ┌──────────┐ │ │ │ 总账凭证 │ │ 申报表 │ │ │ │ 生成 │ │ 输出 │ │ │ └──────────┘ └──────────┘ │ └─────────────────────────────────────────────────────────────────┘ │ ┌───────────────┼───────────────┐ ▼ ▼ ▼ ┌──────────┐ ┌──────────┐ ┌──────────┐ │ 发票管理 │ │ 应收应付 │ │ 报表中心 │ └──────────┘ └──────────┘ └──────────┘ `

3.2 税金管理

#### 3.2.1 增值税管理 ` ┌─────────────────────────────────────────────────────────────┐ │ 增值税计算逻辑 │ ├─────────────────────────────────────────────────────────────┤ │ │ │ 销项税额 = Σ(销售收入 × 税率) │ │ │ │ 进项税额 = Σ(采购成本 × 税率) │ │ + Σ(费用支出 × 税率) │ │ + 海关缴款书税额 │ │ - 进项转出额 │ │ │ │ 可抵扣进项 = MIN(进项税额, 销项税额) │ │ + 加计抵减额（符合条件） │ │ │ │ 应纳增值税 = MAX(销项税额 - 可抵扣进项, 0) │ │ │ │ 留抵税额 = MAX(可抵扣进项 - 销项税额, 0) → 结转下期抵扣 │ │ │ └─────────────────────────────────────────────────────────────┘ ` #### 3.2.2 所得税管理 ` ┌─────────────────────────────────────────────────────────────┐ │ 所得税计算逻辑 │ ├─────────────────────────────────────────────────────────────┤ │ │ │ 收入总额 = 主营业务收入 + 其他业务收入 + 营业外收入 │ │ │ │ 扣除项目 = 营业成本 + 期间费用 + 税金及附加 │ │ + 损失及其他 │ │ │ │ 应税所得额 = 收入总额 - 不征税收入 │ │ - 免税收入 - 扣除项目 │ │ ± 纳税调整项 │ │ │ │ 应纳所得税 = 应税所得额 × 适用税率 │ │ - 减免税额 + 抵免税额 │ │ │ │ 季度预缴 = 应税所得额 × 25% / 4 (一般企业) │ │ │ │ 年度汇算 = 年度应纳所得税 - 已预缴总额 │ │ │ └─────────────────────────────────────────────────────────────┘ ` #### 3.2.3 附加税管理 `python

附加税计算规则

附加税基数 = 增值税应纳税额 + 消费税额 城建税 = 附加税基数 × 城建税税率 # 7%/5%/1% 教育费附加 = 附加税基数 × 3% 地方教育附加 = 附加税基数 × 2%

减免规则

if 小规模纳税人 and 季度销售额 <= 30万: 附加税 = 0 # 免征 ` #### 3.2.4 税金计提方案 | 方案编码 | 方案名称 | 计提规则 | 凭证模板 | |:--------|:--------|:--------|:---------| | TX-001 | 增值税计提 | 应纳增值税×100% | 借: 税金及附加 贷: 应交税费-增值税 | | TX-002 | 附加税计提 | 附加税合计×100% | 借: 税金及附加 贷: 应交税费-附加税 | | TX-003 | 所得税计提 | 应纳所得税×100% | 借: 所得税费用 贷: 应交税费-所得税 | | TX-004 | 个税代扣 | 代扣代缴金额×100% | 借: 应付职工薪酬 贷: 应交税费-个税 |
