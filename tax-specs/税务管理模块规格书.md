## 1. 模块概述

### 1.1 业务定位

税务管理模块是ERP财务系统的**税务数据中心**，负责税务基础配置、税金自动计提、税务申报管理、税负分析等全流程税务管理工作。它与总账、应收应付、发票管理模块深度集成，实现业财税一体化。

### 3.2 税金管理

#### 3.2.1 增值税管理

```
┌─────────────────────────────────────────────────────────────┐
│                    增值税计算逻辑                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  销项税额 = Σ(销售收入 × 税率)                               │
│                                                             │
│  进项税额 = Σ(采购成本 × 税率)                               │
│            + Σ(费用支出 × 税率)                              │
│            + 海关缴款书税额                                  │
│            - 进项转出额                                      │
│                                                             │
│  可抵扣进项 = MIN(进项税额, 销项税额)                        │
│              + 加计抵减额（符合条件）                        │
│                                                             │
│  应纳增值税 = MAX(销项税额 - 可抵扣进项, 0)                  │
│                                                             │
│  留抵税额 = MAX(可抵扣进项 - 销项税额, 0)  → 结转下期抵扣    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 3.2.2 所得税管理

```
┌─────────────────────────────────────────────────────────────┐
│                    所得税计算逻辑                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  收入总额 = 主营业务收入 + 其他业务收入 + 营业外收入          │
│                                                             │
│  扣除项目 = 营业成本 + 期间费用 + 税金及附加                  │
│           + 损失及其他                                        │
│                                                             │
│  应税所得额 = 收入总额 - 不征税收入                          │
│             - 免税收入 - 扣除项目                            │
│             ± 纳税调整项                                      │
│                                                             │
│  应纳所得税 = 应税所得额 × 适用税率                          │
│              - 减免税额 + 抵免税额                          │
│                                                             │
│  季度预缴 = 应税所得额 × 25% / 4  (一般企业)                 │
│                                                             │
│  年度汇算 = 年度应纳所得税 - 已预缴总额                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 3.2.3 附加税管理

```python
# 附加税计算规则
附加税基数 = 增值税应纳税额 + 消费税额

城建税 = 附加税基数 × 城建税税率  # 7%/5%/1%
教育费附加 = 附加税基数 × 3%
地方教育附加 = 附加税基数 × 2%

# 减免规则
if 小规模纳税人 and 季度销售额 <= 30万:
    附加税 = 0  # 免征
```

## 5. 税负分析报表

### 5.1 核心分析指标

| 指标 | 计算公式 | 参考值 |
|:-----|:--------|:-------|
| 增值税税负率 | 本期应纳增值税 / 本期销售收入 × 100% | 1%-3% |
| 所得税税负率 | 本期应纳所得税 / 本期利润总额 × 100% | 15%-25% |
| 综合税负率 | 本期总税金 / 本期销售收入 × 100% | 3%-6% |
| 税前利润率 | 利润总额 / 销售收入 × 100% | 行业参考 |
| 税后利润率 | 净利润 / 销售收入 × 100% | 行业参考 |

### 5.2 风险预警规则

```yaml
风险预警:
  - 等级: 高风险
    条件: 增值税税负率 < 0.5%
    建议: 请检查进项发票是否充分取得

  - 等级: 中风险
    条件: 所得税税负率 < 10%
    建议: 请核实成本费用扣除是否合规

  - 等级: 提醒
    条件: 申报截止日前3天未申报
    建议: 请尽快完成本期税务申报
```

## 6. 数据库设计

### 6.1 核心表结构

#### tax_org（税务组织表）

```sql
CREATE TABLE tax_org (
    id VARCHAR(36) PRIMARY KEY,
    org_code VARCHAR(20) NOT NULL UNIQUE,
    org_name VARCHAR(100) NOT NULL,
    tax_id VARCHAR(20),           -- 纳税人识别号
    taxpayer_type VARCHAR(20),     -- 纳税人类型
    tax_authority VARCHAR(100),    -- 主管税务机关
    filing_cycle VARCHAR(20),      -- 申报周期
    status TINYINT DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

#### tax_rate（税率配置表）

```sql
CREATE TABLE tax_rate (
    id VARCHAR(36) PRIMARY KEY,
    tax_type VARCHAR(20) NOT NULL,     -- 税种
    tax_name VARCHAR(50) NOT NULL,      -- 税种名称
    rate DECIMAL(8,4),                  -- 税率
    rate_type VARCHAR(20),              -- 税率类型
    effective_date DATE,                -- 生效日期
    expiry_date DATE,                   -- 失效日期
    status TINYINT DEFAULT 1
```
