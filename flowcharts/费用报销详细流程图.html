📋 凭证模板汇总

EX01 日常费用报销
借 管理费用 + 进项税 → 贷 银行存款
EX02 差旅费报销
借 销售/管理费用-差旅 + 进项税 → 贷 银行存款
EX03 招待费报销
借 管理费用-招待费 → 贷 银行存款
EX04 借款发放
借 其他应收款-员工借款 → 贷 银行存款
EX05 借款核销
借 管理费用 → 贷 其他应收款 + 银行存款
EX06 超标特批
借 管理费用-标准内 + 营业外支出-超标 → 贷 银行存款
EX07 费用分摊
借 多成本中心管理/销售费用 → 贷 银行存款

📊 审批链路对照表

费用类型金额区间审批链路级数
日常费用2千以下部门主管 → 财务审核2级
日常费用2千~1万部门主管 → 部门经理 → 财务审核3级
日常费用1万以上部门主管 → 部门经理 → 分管副总 → 财务审核4级
差旅费用不限部门主管 → 部门经理 → 财务审核3级
招待费用不限部门主管 → 部门经理 → 分管副总 → 财务审核4级
借款申请不限部门主管 → 财务审核 → 出纳确认3级
