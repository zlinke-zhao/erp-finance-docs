## 1. 模块定位与业务目标

### 1.1 一句话定义

**成本核算模块是ERP系统中"归集-分摊-核算-结转"闭环的成本管控引擎，负责从直接材料/直接人工/制造费用归集开始，经过多维度分摊、在制品核算、完工产品成本计算，最终生成成本结转凭证写入总账的全流程管理。**

### 1.2 在整体流程中的位置

```
┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────────────┐
│ 采购入库  │─▶│ 生产领料  │─▶│ 工时采集  │─▶│ 费用报销归集      │
│ 材料成本  │  │ 直接材料  │  │ 直接人工  │  │ 制造费用          │
└──────────┘  └──────────┘  └──────────┘  └────────┬─────────┘
                                                       │
                                                       ▼
                               ┌─────────────────────────────────────────┐
                               │  🏭 成本核算管理（本模块）                │
                               │                                         │
                               │  F1 成本归集    →  材料+人工+制费三要素   │
                               │  F2 制费分摊    →  多动因分摊到成本中心   │
                               │  F3 在制品核算  →  约当产量法/定额法     │
                               │  F4 完工结转    →  产品成本计算+结转     │
                               │  F5 差异分析    →  标准/实际差异对比      │
                               │  F6 成本报表    →  成本明细+趋势+结构    │
                               │  F7 凭证生成    →  自动凭证+成本归集      │
                               └─────────────────┬───────────────────────┘
                                                 │
                      ┌──────────────────────────┼──────────────────────┐
                      ▼                          ▼                      ▼
              ┌──────────────┐         ┌──────────────┐       ┌──────────────┐
              │ 🧾 总账凭证    │         │ 📊 成本分析    │       │ 📋 存货核算    │
              │ CT01~CT08    │         │ 差异+趋势     │       │ 产成品入库    │
              └──────────────┘         └──────────────┘       └──────────────┘
```

### 1.3 与已有模块的衔接关系

| 上游模块 | 衔接点 | 本模块消费的数据 |
|:---|:---|:---|
| **采购入库** | 材料入库成本 | 采购单价、运杂费、关税 |
| **生产领料** | 直接材料归集 | 领料单号、物料、数量、单价 |
| **工时管理** | 直接人工归集 | 工时记录、人工费率 |
| **费用报销模块** | 制造费用归集 | 车间/工厂相关费用 |
| **固定资产模块** | 折旧费归集 | 设备折旧、按部门分摊 |
| **存货核算** | 期初期末在制品 | 在制品数量、完工程度 |

| 下游模块 | 衔接点 | 本模块产出的数据 |
|:---|:---|:---|
| **总账凭证引擎** | 成本结转 → 自动凭证 | CT01~CT08 凭证模板触发事件 |
| **存货核算** | 完工产品入库 | 产品单位成本、入库数量 |
| **销售出库** | 销售成本结转 | 产品成本、出库成本 |
| **成本分析** | 多维度分析 | 实际成本、标准成本、差异 |

### 1.4 核心业务目标

| 目标 | 指标 | 当前痛点 |
|:---|:---|:---|
| **成本核算及时性** | 月结后3个工作日内完成成本核算 | 手工统计耗时2周，数据滞后 |
| **成本准确性** | 实际成本与标准成本差异率不超过5% | 分摊基准不合理，成本失真 |
| **在制品核算** | 在制品估值误差不超过3% | 约当产量手工估算，偏差大 |
| **成本可追溯** | 每笔成本可追溯到原始单据 | 费用归集链路断裂，无法追溯 |
| **差异可分析** | 量差/价差自动拆分 | 只看总额差异，无法定位原因 |

## 6. 关键业务规则

### 6.1 成本核算方法

| 方法 | 适用场景 | 核算逻辑 |
|:---|:---|:---|
| **品种法** | 大量单步骤生产 | 按产品品种归集+分配 |
| **分批法** | 单件小批生产 | 按生产订单/批号归集 |
| **分步法** | 大量多步骤生产 | 按生产步骤逐步结转 |
| **作业成本法** | 间接费用占比高 | 按作业动因分摊制费 |

### 6.2 在制品核算方法

| 方法 | 计算逻辑 | 适用场景 |
|:---|:---|:---|
| **约当产量法** | 期末在制品数量 x 完工程度 = 约当产量，再按比例分配成本 | 均匀投料、加工进度可测 |
| **定额成本法** | 期末在制品按定额成本计价，实际成本与定额差异由完工产品承担 | 定额管理完善 |
| **不计算在制品** | 当期发生的成本全部由完工产品承担 | 在制品数量少或稳定 |

### 6.3 制造费用分摊基准

| 分摊基准 | 计算方式 | 适用场景 |
|:---|:---|:---|
| **生产工时** | 制费总额 x 某产品工时/总工时 | 人工密集型 |
| **机器工时** | 制费总额 x 某产品机时/总机时 | 设备密集型 |
| **直接材料成本** | 制费总额 x 某产品材料/总材料 | 材料相关制费 |
| **直接人工成本** | 制费总额 x 某产品人工/总人工 | 人工相关制费 |
| **产品产量** | 制费总额 x 某产品产量/总产量 | 单一产品或类似产品 |
| **固定比例** | 按预设比例分摊 | 无法确定单一基准 |

### 6.4 成本结转顺序

```
1. 材料费用归集 → CT01
2. 人工费用归集 → CT02  
3. 制造费用归集 → CT03
4. 辅助生产分配 → CT05
5. 制造费用分摊 → CT04
6. 在制品核算 → 约当产量计算
7. 完工产品入库 → CT06
8. 销售成本结转 → CT07
9. 成本差异处理 → CT08
10. 月末结账
```

> ⚠️ 结转顺序不可逆，前一步未完成不可执行下一步

### 6.5 差异分析公式

| 差异类型 | 计算公式 |
|:---|:---|
| **材料量差** | (实际用量 - 标准用量) x 标准单价 |
| **材料价差** | (实际单价 - 标准单价) x 实际用量 |
| **人工效率差异** | (实际工时 - 标准工时) x 标准工资率 |
| **工资率差异** | (实际工资率 - 标准工资率) x 实际工时 |
| **制费耗费差异** | 实际制费 - 预算制费(按实际产量调整) |
| **制费效率差异** | (实际工时 - 标准工时) x 标准分摊率 |

## 7. 状态机设计

### 7.1 成本核算期间状态

```
未开始 ──材料归集──▶ 材料已归集 ──人工归集──▶ 人工已归集
  │                                               │
  └──────全部归集完成后──▶ 制费已归集 ──制费分摊──▶ 制费已分摊
                                                        │
                              辅助生产分配 ──▶ 在制品核算 ──▶ 完工结转
                                                              │
                                                    销售成本结转 ──▶ 差异处理 ──▶ 已结账
```

### 7.2 成本池状态流转

```
归集中 ──审核确认──▶ 已审核 ──执行分摊──▶ 已分摊 ──成本结转──▶ 已结转
  │                    │
  └──撤销──▶ 归集中     └──撤销审核──▶ 归集中
```

## 8. 非功能性需求

| 类别 | 要求 |
|:---|:---|
| **性能** | 成本计算(单产品)不超过5秒，全量分摊计算不超过3分钟 |
| **并发** | 月末结账时支持多人同时查询，但结账操作需加锁 |
| **审计** | 所有归集/分摊/结转操作全程留痕，不可删除 |
| **安全** | 成本数据按部门权限隔离，非授权人员不可查看 |
| **可用性** | 分摊试算可反复执行不影响正式数据 |
| **集成** | 对接生产领料、工时管理、费用报销、固定资产折旧等上游模块 |
